Sales Analytics Project 📊

Overview:

This project demonstrates end-to-end sales analytics using SQL, Excel, and Power BI. It covers data cleaning, reporting, dashboard creation, and trend forecasting.

-------------------------------------------------------------------------------------------------------------------------------------------------------------------
Project Structure:

1. Business Analysis & Metrics -
* Defined key sales metrics and KPIs
* Identified funnel bottlenecks and performance indicators

-------------------------------------------------------------------------------------------------------------------------------------------------------------------

2. Data Cleaning & Reporting (SQL + Excel) -
* Cleaned raw CRM data using SQL and Excel
* Standardized deal stages and handled missing values
* Generated reports:
* Overall Sales Performance:
      * KPI summary (Revenue, Deals, Win Rate, Avg Deal Size)
      * Revenue trend with forecasting
      * Sales performance by product and agent
      * Funnel conversion analysis

* Added slicers and tooltips for interactivity

-------------------------------------------------------------------------------------------------------------------------------------------------------------------

Key Insights:

* Revenue is driven by a few top-performing products and sales agents

* Conversion rate indicates opportunity for improvement

* Sales performance shows variability across dimensions

-------------------------------------------------------------------------------------------------------------------------------------------------------------------

Tools Used: 

* SQL (PostgreSQL)

* Microsoft Excel

* Power BI
